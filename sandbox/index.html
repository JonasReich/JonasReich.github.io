Aeronauts

Field of work: Game Design, UX Design, UI Programming
Technologies: Unity3D, C#

Aeronauts is an Android strategy/building game in which you gather resources to manage expeditions to neighbouring floating islands. The game was created during the practical phase of my 2nd semester in a joint effort of 2nd and 3rd semester sutdents.

The main island is where all the buildings are located.

When the game is first started, you are greeted by an Aeronaut worker and introduced to the core game mechanics.

You can construct various buildings on your main island.

You can start expeditions to neighbouring dungeon islands.

Expeditions are the key to a steady supply of resources.

There is a multitude of dungeon islands to explore.

At times you're required to engage in fights versus dangerous insectuous creatures.
